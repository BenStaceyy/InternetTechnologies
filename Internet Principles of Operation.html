Modem

A modulator-demodulator, or simply a modem, is a device that sends information between the outside world and your home. A modem turns your inbound connection into an Ethernet connection, which then allows a Wi-Fi router to connect to the Internet.

Internet Service Provider

A company that provides access to the internet for a fee.

Web server

A web server is software and hardware that uses HTTP and other protocols to respond to client requests made over the World Wide Web. Hosts websites

Proxy server

An intermediary server separating the user and the destinations that they are browsing.
