Malware

Malware is software that is designed to damage or control computer systems. Malware is a contraction for malicious software. Examples of common malware includes viruses, worms, Trojan viruses, spyware, adware, and ransomware.

Virus

A computer virus is a type of computer program that, needs user input to replicate itself. It replicates itself modifying other computer programs and inserting its own code. If this replication succeeds, the affected areas are then said to be "infected" with a computer virus.

Spyware

Spyware is software that can gather data about an individual. Usually broken into four categories adware, system monitors, tracking including and trojans. Examples are keyloggers, rootKits, or web beacons.
